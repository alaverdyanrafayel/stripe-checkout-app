# Message-in-a-Bottle-Core-App

## Config

```console
cp .env.sample .env  // Prepare environment variables for server
cd client && cp .env.sample .env // Prepare environment variables for client
```

## Setup

Install the project.

```console
npm i
```

### Run in development mode

```
npm run start:dev
```
### Run in production mode

```
npm run build
npm start
```
